# Data Description

This project uses dataset provided for internal competition purposes.

Due to data usage restrictions, the dataset cannot be publicly shared.


## Data Structure

The dataset includes:

- Environmental variables (Temperature, CO₂, VPD, etc.)
- Physiological indicators (Chl, Fv/Fm, PI_abs, etc.)
- Target variables (Leaf/Root TPC, TFC)
- Scenario labels (SSP1, SSP3, SSP5)


## Note

All code is provided for reproducibility.
